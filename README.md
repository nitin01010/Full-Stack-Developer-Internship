# 🎬 AI Movie Insight Builder — Frontend

## 📌 Overview

AI Movie Insight Builder is a modern React-based frontend application where users can enter an **IMDb Movie ID** and view movie details along with AI-generated insights.

This project is built as part of a **Full-Stack Developer Internship Assignment**.

---

## ✨ Features

### 🎥 Movie Information
- Search movie using IMDb ID
- Display:
  - Movie title
  - Poster
  - Release year
  - IMDb rating
  - Cast list
  - Plot summary
  - External ratings

### 🤖 AI Summary
- Generate AI-based movie insight
- Audience sentiment summary
- ChatGPT-style response UI

### 🎨 Modern UI
- Clean modern design
- Responsive layout (Desktop + Mobile)
- Smooth animations
- Gradient action buttons
- Card-based layout

---

## 🛠️ Tech Stack

- **React.js**
- **Tailwind CSS**
- **Fetch API**
- **Node.js Backend API**
